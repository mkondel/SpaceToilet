Space Toilet

It's a game! Based on the classic shooter tutorial from Unity. The name comes from the way that ship looks like a toilet seat from the top...

Downloads

Mac version. ZIP file (54MB)
Linux version. ZIP file (55MB)

Note that on linux need to set resolution before playing 1st time. To do this, hold the Alt key when starting the game. Set your resolution to the one closest to 900x600.

Android version. APK file (49MB)
Windows version. ZIP file (73MB)
iOS version. ZIP file (190MB)

Note that you must have a jailbroken ipad/iphone. Here are some iOS screenshots though:)
